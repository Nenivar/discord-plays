# discord-plays
Play pokemon over discord with commands

# Installation
Requires:
* Linux (probably)
* xdotool
```
apt-get install xdotool
```

Store token in auth.json:
```json
{
    "Client_ID" : "",
    "Bot_Token" : ""
}
```
Change stuff in config.json

Load (legally obtained) ROM -> VBA-M

```
bundle exec ruby ping.rb
```
